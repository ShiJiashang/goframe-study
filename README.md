# GoFrame 实战学习

这是一个按课次逐步推进的 GoFrame v2.10 学习仓库。前 6 课是彼此独立的小样例，
第 7 课开始构建轻量商城 API。

完整学习计划见：[STUDY_PLAN.md](STUDY_PLAN.md)。

## 环境

- Go 1.26.3
- GoFrame v2.10.2
- GoFrame CLI v2.10.2：`gf`
- 后续课程使用 Docker MySQL 8 与 Redis 7

如果终端提示 `gf: command not found`，请先把 Go bin 目录加入 PATH。
